# Debugging

With so many moving pieces involved in an end-to-end test, one must know their debugging tools. These are going to come in handy for both your local test runs as well as those that will be running remotely on CI.

In this exercise block, you are going to equip yourself with two essential debugging techniques that I've been personally using to a great success when tinkering with my Playwright tests:

1. UI mode;
1. Trace viewer.

As a bonus, I have prepared for you a focused practice exercise in the "Extras" where you will get your hands dirty with Live debugging in Playwright. Ready? Let's go hunt some bugs!
